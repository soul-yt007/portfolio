# Chandradeep Singh Rathore — Personal Portfolio

A modern, dark-themed personal portfolio website showcasing my work, skills, and journey as a coder, developer, entrepreneur, editor, gamer, and photographer.

---

## About the Project

This is my personal portfolio website built to represent who I am and what I do.  
It highlights my background, work experience, side projects, and contact details in a clean and futuristic design.

---

## Features

- Dark futuristic UI with neon glow effects  
- Fully responsive for all devices  
- Clean and simple single-page layout  
- Custom CSS animations and styling  
- Direct contact links (Call, Email, Instagram, LinkedIn)

---

## Tech Stack

- HTML5  
- CSS3  
- Google Fonts (Fira Code)
